Comment héberger un site web gratuitement et de manière permanente avec GitHub et Cloudflare Pages


Introduction

Dans le monde du développement web, l'hébergement de votre site web est une étape cruciale. Si vous recherchez un moyen de le faire gratuitement
et de manière permanente, GitHub et Cloudflare Pages sont des outils fantastiques à considérer. Dans cet article, nous allons vous montrer
étape par étape comment accomplir cela.

Étape 1 : Héberger vos fichiers sur GitHub

Pour commencer, assurez-vous d'avoir un compte GitHub. Ensuite :

  1. Créez un nouveau dépôt GitHub pour votre site web.
  2. Téléchargez vos fichiers HTML, CSS, JavaScript, etc., dans ce dépôt.
  3. Assurez-vous que votre fichier principal s'appelle index.html pour la page d'accueil.
  4. Publiez vos fichiers sur GitHub.

Étape 2 : Configurer Cloudflare Pages

Cloudflare Pages vous permet de déployer votre site web en quelques étapes simples :

  1. Connectez votre compte GitHub à Cloudflare Pages.
  2. Sélectionnez le dépôt que vous avez créé précédemment.
  3. Choisissez la branche que vous souhaitez déployer.
  4. Configurez les paramètres de construction, si nécessaire.
  5. Appuyez sur le bouton de déploiement.

Étape 3 : Configurer votre nom de domaine (facultatif)

Si vous avez un nom de domaine, vous pouvez le configurer avec Cloudflare pour qu'il pointe vers votre site hébergé. Cela donne à votre site une URL personnalisée.

Conclusion

En suivant ces étapes, vous pouvez héberger votre site web gratuitement et de manière permanente en utilisant GitHub et Cloudflare Pages.
Vous bénéficiez ainsi de la stabilité de GitHub pour le stockage des fichiers et de la puissance de Cloudflare pour la mise en ligne de votre site.

Merci d'avoir lu cet article. Nous espérons que cela vous a été utile pour héberger votre site web avec succès !
